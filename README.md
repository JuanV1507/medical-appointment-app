
# Sistema de Citas Médicas

## Objetivo general
Desarrollar una aplicación web para la gestión de citas médicas que permita administrar pacientes, médicos y citas de manera eficiente.

## Tecnologías a utilizar
- Laravel 12
- PHP 8.2
- MySQL
- Livewire
- Jetstream
- Node.js
- GitHub
